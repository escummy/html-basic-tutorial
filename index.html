What exactly is HTML?


HyperText Markup Language

It is the most basic component of the Web. Define the meaning and structure of web content by using "markup" to tag text, images, and other content for display in a web browser.

All tags in HTM have an opening symbol and a closing symbol except for some specific cases that you will see in the tutorial.

Before starting, we must clarify that the 'index.html' document refers to the main document of a website, it will make a request to a local server in which it will display the labels that we will see on the screen.


** IMPORTANT **

HTML document is not to make it look good, it is simply used to give the browser a series of information regarding the structure of our website, to make it more beautiful or aesthetic instead, we will always use 'Cascade Style Sheet', and although in some cases it may be necessary to do it in the HTML document, it's not recommended at all to give styles here, we have already CSS language to do that.
